# 小红书内容规范详解

## 标题规范

### 长度限制
- **最大长度**：20 个字（中文字符）
- **建议长度**：15-18 个字
- **超出处理**：自动截断或发布失败

### 标题公式

**1. 数字法**
```
格式：数字 + 动作 + 结果
示例：
- 3分钟做完8小时工作
- 5个技巧让效率翻倍
- 10款工具只推荐这1个
```

**2. 对比法**
```
格式：从A → B
示例：
- 从1小时→3分钟
- 从小白→大神
- 从0粉→10万粉
```

**3. 痛点法**
```
格式：痛点 + 解决方案
示例：
- 被老板夸了3次的秘密
- 终于不用加班了
- 摸鱼神器来了
```

**4. 疑问法**
```
格式：为什么/怎么办/如何
示例：
- 为什么AI写的比我火？
- 如何3天涨粉1000？
- 怎么让老板主动加薪？
```

### 禁用词汇

**绝对化用语**（严禁）：
- 最、第一、唯一、必须、一定
- 100%、完美、绝对、永久
- 顶级、极致、终极、史上

**替代方案**：
- "最好" → "超好用"
- "第一" → "排名靠前"
- "100%" → "几乎全部"
- "必须" → "建议"

## 正文规范

### 长度限制
- **最大长度**：1000 个字
- **建议长度**：600-800 个字
- **最小长度**：200 个字

### 内容结构

**标准模板**：
```
1. 痛点开头（50-100字）
   - 描述用户痛点
   - 引发共鸣

2. 效果对比（100-150字）
   - 使用前 vs 使用后
   - 数据对比

3. 3步教程（400-500字）
   - Step 1: 具体操作
   - Step 2: 具体操作
   - Step 3: 具体操作

4. 互动引导（50-100字）
   - 评论区互动
   - 收藏提醒

5. 标签（最后一行）
   - #标签1 #标签2 #标签3
```

### 段落要求
- **每段长度**：50-100 字
- **段落间隔**：空一行
- **使用表情**：适当使用 emoji
- **使用列表**：便于阅读

### 禁止内容

**1. 诱导互动**（严禁）：
- ❌ "评论区回复XX"
- ❌ "点赞关注领资料"
- ❌ "转发抽奖"
- ✅ "评论区告诉我"
- ✅ "收藏起来慢慢看"

**2. 拉踩行为**（严禁）：
- ❌ 多品牌对比
- ❌ 贬低竞品
- ❌ "XX不如YY"
- ✅ 只介绍自己推荐的

**3. 虚假宣传**（严禁）：
- ❌ 夸大效果
- ❌ 虚假数据
- ❌ 误导性描述

## 封面规范

### 尺寸要求
- **标准尺寸**：1080x1440（3:4）
- **最小尺寸**：800x1066
- **文件大小**：≤5MB
- **格式**：JPG、PNG

### 设计要求

**1. 无留白**
- 填满整个画布
- 不要底部留白
- 不要四周留白

**2. 高对比度**
- 黄黑、红黑、蓝白
- 文字清晰可读
- 背景不抢眼

**3. 文字排版**
- 大字标题（60-80pt）
- 简洁副标题（30-40pt）
- 最多 2 行文字

**4. 视觉冲击**
- 吸引眼球
- 突出重点
- 简洁明了

### 禁止内容
- ❌ 二维码
- ❌ 联系方式
- ❌ 外部链接
- ❌ 低俗内容
- ❌ 侵权图片

## 标签规范

### 数量限制
- **最大数量**：10 个
- **建议数量**：5-8 个
- **最小数量**：3 个

### 标签选择

**1. 核心标签**（必选）
```
#OpenClaw
#AI工具
#效率提升
```

**2. 场景标签**（选 2-3 个）
```
#职场技巧
#副业赚钱
#学习方法
#生活技巧
```

**3. 热门标签**（选 1-2 个）
```
#今日热点
#热门话题
#爆款推荐
```

### 标签格式
- **格式**：`#标签名`
- **位置**：正文最后一行
- **间隔**：空格分隔
- **示例**：`#OpenClaw #AI工具 #效率提升`

### 标签策略

**1. 精准匹配**
- 与内容高度相关
- 不要堆砌无关标签

**2. 热度平衡**
- 2-3 个热门标签（竞争大）
- 3-4 个中等标签（精准流量）
- 1-2 个小众标签（垂直用户）

**3. 定期更新**
- 每周检查标签热度
- 替换低效标签
- 测试新标签

## 发布时间

### 最佳时间段

**工作日**：
- 12:00-13:00（午休）
- 18:00-19:00（下班前）
- 20:00-22:00（晚间黄金时段）

**周末**：
- 10:00-12:00（上午）
- 14:00-16:00（下午）
- 20:00-22:00（晚间）

### 避免时间
- 凌晨 0:00-6:00
- 上午 9:00-11:00（工作时间）
- 下午 14:00-17:00（工作时间）

## 违规处理

### 违规等级

**轻度违规**：
- 限流 7 天
- 删除违规内容
- 警告通知

**中度违规**：
- 限流 30 天
- 删除违规内容
- 扣除信用分

**重度违规**：
- 永久限流
- 封禁账号
- 无法申诉

### 预防措施

1. **发布前检查**
   - 运行违规词检测
   - 人工复核内容
   - 检查封面合规

2. **定期自查**
   - 每周检查历史内容
   - 删除风险内容
   - 更新违规词库

3. **学习规则**
   - 关注平台公告
   - 学习案例分析
   - 参加官方培训

## 内容优化

### 数据指标

**核心指标**：
- 阅读量（曝光）
- 点赞率（内容质量）
- 收藏率（实用性）
- 评论率（互动性）

**优化目标**：
- 点赞率 > 5%
- 收藏率 > 3%
- 评论率 > 1%

### 优化策略

**1. 标题优化**
- A/B 测试不同标题
- 分析高点击标题特征
- 建立标题模板库

**2. 封面优化**
- 测试不同设计风格
- 分析高点击封面特征
- 建立封面模板库

**3. 内容优化**
- 分析爆款内容结构
- 优化教程步骤
- 增加互动元素

**4. 标签优化**
- 测试不同标签组合
- 分析标签带来的流量
- 优化标签策略
