# 小红书违规词列表

## 绝对化用语（严禁）

### 程度词
- 最、最佳、最好、最优、最强
- 第一、唯一、独一无二
- 顶级、极致、终极、巅峰
- 完美、绝对、永久、永恒
- 史上、全网、全球、世界

### 效果词
- 100%、必须、一定、肯定
- 保证、确保、承诺、担保
- 立即、马上、瞬间、秒
- 彻底、根治、永不、永远

### 替代方案
| 违规词 | 替代词 |
|--------|--------|
| 最好 | 超好用、很不错 |
| 第一 | 排名靠前、名列前茅 |
| 100% | 几乎全部、大部分 |
| 必须 | 建议、推荐 |
| 完美 | 很棒、优秀 |
| 永久 | 长期、持久 |
| 顶级 | 高端、优质 |
| 极致 | 出色、卓越 |

## 检测工具

### 自动检测
使用脚本自动检测违规词：

```python
def check_violations(text):
    """检测违规词"""
    violations = []
    
    # 绝对化用语
    absolute_words = ['最', '第一', '唯一', '100%', '必须', '完美']
    for word in absolute_words:
        if word in text:
            violations.append(f"绝对化用语: {word}")
    
    # 诱导互动
    interaction_words = ['评论区回复', '点赞关注', '转发抽奖']
    for word in interaction_words:
        if word in text:
            violations.append(f"诱导互动: {word}")
    
    return violations
```
